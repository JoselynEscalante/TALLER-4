HTML

HTML es el lenguaje que se emplea para el desarrollo de páginas de internet.
Está compuesto por una seríe de etiquetas que el navegador interpreta y da forma en la pantalla.
HTML dispone de etiquetas para imágenes, hipervínculos que nos permiten dirigirnos a otras páginas, saltos de línea, listas, tablas, etc.

¿Cuáles son las etiquetas HTML básicas?

Title: Se utiliza para definir un título para el documento.
Body: Define la parte corporal o cuerpo de un documento .
P:Define un trozo de texto como párrafo.
Strong:Define un texto concreto en negrita.
Table:Permite la creación o definición de una tabla.
Img src:Permite insertar o definir una imagen personalizada.

Etiqueta Descripción Ejemplo
<b> Texto en negrita <b>Joselyn Escalante </b>
<strong> Texto importante <strong>Joselyn Escalante</strong>
<i> Texto en cursiva <i>Joselyn Escalante</i>
<p> Inserta un párrafo <p>Joselyn Escalante</p>
<img> Inserta una imagen <img src="imag1.jpg" width="10" height="10"/>
<small> Letra pequeña <small>Joselyn Escalante</small>
<mark> Resalta la palabra <mark>Joselyn Escalante</mark>
