HMTL

HTML es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

Versiones de HTML

Etiquetas y atributos

Las etiquetas son la estructura básica del HTML. Estas etiquetas se componen y contienen otras propiedades, como son los atributos y el contenido. HTML define un total de 91 etiquetas, de las cuales 10 se consideran obsoletas. A continuación veremos algunas etiquetas mas usadas en clase:

Etiqueta Descripción Ejemplo
<b> Texto en negrita <b>Joselyn Escalante </b>
<strong> Texto importante <strong>Joselyn Escalante</strong>
<i> Texto en cursiva <i>Joselyn Escalante</i>
<p> Inserta un párrafo <p>Joselyn Escalante</p>
<img> Inserta una imagen <img src="imag1.jpg" width="10" height="10"/>
<small> Letra pequeña <small>Joselyn Escalante</small>
<mark> Resalta la palabra <mark>Joselyn Escalante</mark>

Estructura HTML

HTML:

Esta al inicio de un documento HTML e indica a los navegadores que la página tiene código HTML, para que pueda leerlo de esa forma.


ENCABEZADO:(HEAD)

Es la etiqueta que se utiliza para el encabezado de la página. Su principal función es contener toda la información del funcionamiento del sitio. Debido a esto, es un código encriptado que las personas que entran en la página no pueden ver.


TÍTULO DE LA PÁGINA:(TITLE)

Es la etiqueta que da a un sitio un nombre para que los usuarios puedan identificarlos. Es el título que puedes ver en las pestañas del navegador.


CUERPO:(BODY)

Es la etiqueta que contiene todos los elementos individuales del sitio. O dicho de otra forma, comprende todo el contenido visible del sitio. Aquí podras insertar texto, imágenes , videos o cualquier otra funcionalidad que desees mostrar.


CSS - CLASE

Una clase es la descripción de un conjunto de objetos similares; consta de métodos y datos que resumen las caracteristicas comunes de dicho conjunto.

CSS - SINTAXIS

la base del CSS es lograr un estilo homogéneo y facil de modificar para todo nuestro sitio. Por ello es importante exportar los estilos a una hoja de estilos, (un archivo de extensión .css) Esta opcion resulta mucho mejor , ya que nos permite centralizar ese estilo en un solo archivo que aplicaremos a todas las páginas.
